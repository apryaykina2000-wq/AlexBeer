# AlexBeer (Runner)
Игра-раннер на Phaser 3. Цель — собрать 60 предметов (пиво/рыба/бокал).

## Управление
Клик/тап — прыжок.

## Медали
- 20 очков — Бронза
- 40 очков — Серебро
- 60 очков — Золото + конфетти

## Запуск локально
Откройте `index.html` с любого статического сервера (например, VSCode Live Server).
Для GitHub Pages просто залейте файлы в репозиторий и включите Pages.
